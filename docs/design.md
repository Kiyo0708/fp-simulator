# FP Simulator 設計書（たたき台）

最終更新: 2026-04-27

---

## 1. アプリ概要

一般ユーザーが自分でライフプランを入力し、年齢ごとの資産・収支推移をグラフで確認できるセルフサービス型FPシミュレーター。

---

## 2. 画面構成・遷移

```
トップページ
  └─ [プランを作成する] ─→ 結果ページ（メインハブ）
       ├─ 左サイドバー各メニュー → 各入力ページ
       │    ├─ 家族構成
       │    ├─ 収入
       │    ├─ 支出（生活費・住宅）
       │    └─ お子さま関連
       └─ [ステップで入力] ─→ ウィザード形式（家族→収入→支出→子供→結果）
```

- **結果ページがメインハブ**。各入力ページから戻ると結果グラフがリアルタイム更新される
- 入力ページは左サイドバーから直接ジャンプも、ウィザードで順番に進むことも可能

---

## 3. 入力項目

### 3-1. 家族構成

| 項目 | 内容 |
|---|---|
| 家族代表者 | 名前（任意）・現在年齢・性別 |
| 配偶者 | 名前（任意）・現在年齢・性別・有無 |
| 子ども | 人数・各子どもの現在年齢（最大6人） |

### 3-2. 収入

| 項目 | 内容 |
|---|---|
| 月収（手取り） | 家族代表者 |
| 月収（手取り） | 配偶者（いる場合） |
| 積み立て額 | 毎月の投資・貯蓄額 |
| 想定利回り | 年率（%）|

> 将来拡張：退職金・年金受給開始年齢・年金受給額

### 3-3. 支出

| カテゴリ | 項目 |
|---|---|
| 生活費 | 月々の生活費（食費・光熱費・通信費など） |
| 住宅関連 | **v1: 月々の家賃**　／　v2以降: 住宅ローン（頭金・借入額・返済期間・金利） |
| 固定支出 | その他の月々固定費（保険料など） |

### 3-4. お子さま関連（子ども1人につき）

各学校フェーズで「公立 / 私立 / 進学しない」を選択 → 費用が自動計算される。

| フェーズ | 年齢目安 | 選択肢 |
|---|---|---|
| 保育園 | 0〜3歳 | 認可 / 認可外 / 入所しない |
| 幼稚園 | 4〜6歳 | 公立 / 私立 / 入園しない |
| 小学校 | 7〜12歳 | 公立 / 私立 |
| 中学校 | 13〜15歳 | 公立 / 私立 |
| 高校 | 16〜18歳 | 公立 / 私立 / 進学しない |
| 大学・大学院 | 19〜22歳 | 国公立 / 私立文系 / 私立理系 / 専門学校 / 進学しない |
| 自宅通学 or 一人暮らし | 大学期間 | 自宅 / 賃貸（生活費加算） |

費用の単価は文部科学省の調査データをベースにしたデフォルト値を設定。手動上書きも可能。

## 4. 結果画面

### 4-1. グラフ

- **積み上げ棒グラフ**（横軸: 年齢、縦軸: 万円）
- 家族全員の年齢を横軸下部に表示（GLiP参考）
- 表示切り替え: 収入 / 支出 / 資産残高
- 凡例でカテゴリ別（生活費・住宅・子ども・老後など）のON/OFF切り替え
- **資産残高がマイナスになった年の棒グラフ・数字を赤で表示**

### 4-2. 年次サマリー表

グラフ下に年齢ごとの内訳テーブルを表示。

| 年齢 | 収入 | 支出 | 貯蓄 | 資産残高 |
|---|---|---|---|---|
| 35歳 | 600万 | 480万 | 120万 | 850万 |
| 36歳 | 600万 | 510万 | 90万 | 940万 |
| ... | | | | |

資産残高がマイナスのセルは赤字表示。

## 5. 保存・読み込み

- **ダウンロード**: 入力内容を JSON ファイルとして保存
- **アップロード**: JSON ファイルを読み込み、前回の入力を復元してグラフを再描画
- サーバーへの保存・アカウント管理は行わない（クライアントサイドのみ）

```json
// 保存JSONの構造イメージ
{
  "version": "1.0",
  "savedAt": "2026-04-27T00:00:00Z",
  "family": { ... },
  "income": { ... },
  "expenses": { ... },
  "children": [ ... ]
}
```

## 6. 実装優先順位

| フェーズ | 内容 |
|---|---|
| v1（MVP） | 家族構成・収入・生活費・家賃の入力 → 資産推移グラフ + テーブル → JSON保存/読み込み |
| v2 | 子ども教育費の入力・計算 |
| v3 | 住宅ローン対応（賃貸 or 購入の比較） |
| v4 | 退職・年金・老後プランの入力 |
| v5 | PDF/CSVエクスポート |
